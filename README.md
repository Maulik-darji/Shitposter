## Text-only microblog UI (X-inspired patterns, not a clone)

This repo contains a lightweight, **text-only** microblogging web UI built in **React (non‑Vite)** with a 3‑column “timeline” layout (nav / feed / sidebar).

### Run

Install deps:

```bash
npm install
```

Start dev server:

```bash
npm start
```

Build:

```bash
npm run build
```

### Notes

- Posts are **text-only** (no images/videos/uploads).
- Data persists in your browser via `localStorage` (prototype).
